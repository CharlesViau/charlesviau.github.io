About The Project

Description

Get It & Get Out is a VR game that feature a grapple hook mechanic. You have to collect treasure in a city located inside a volcano before the lava reaches the highest point. The game other features involve: destruction, 3D ambiant sounds, minimal UI with VR interactions as well as flocking AI.

Technologies

Language
C#
Software & Ressources
Unity, Rider, Wwise, Pro Tools
Unity Specifics
XR toolkit, Wwize API

My Role

In this project, we were a team of five. My role in this project was mainly to learn the Wwise software and the API to integrate it into Unity. I pass the Audiokinetick Wwise certification 101 and 301 so far. I had to create all the sound using prior knowledge I had acquired in sound design to create all sound asset, create the Wwise project and write the proper code in unity to call events, as well as code unique audio behavior for ambiance, treasure proximity, etc.. I also had to optimize the sound as much as I could for VR and android. Other than that, I also worked on designing the core architecture of the project.
