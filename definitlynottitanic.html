About The Project

Description

This Project was made in 48 hours during Ludum Dare 50th in the first weekend of April 2022. The theme of the game jam was ''delay the inevitable'' and the goal of our game is to patch holes with woods to stall for time before the ship sinks into the sea. You can see the comments and review on the jam page of our game by clicking here.

Technologies

Language
C#
Software & Ressources
Unity, Rider, Pro Tools, Finale, NotePerformer, MagicaVoxel
Unity Specifics
UI, Animator, Audio

My Role

For this project we were a three man squad and it was a great experience because I had to teach Unity to someone who had never touch it before, so, it was a nice opportunity to work on my communication and explanation skills of programming concepts. Our game features an opening and an ending cutscene using unity Timeline. In this jam, since I was not only focusing on the audio, but more on gameplay and UI, I had something like 4h hours to come up with everything that is audio related(sound design, music composition) and implement it.
