About The Project

Description

Heavenly Hand - Galactic Savior is a tower defense game. You have to defend a dyson ring against multiple waves of asteroids until you lose by placing different towers, upgrading them and placing planets or black hole to help clearing them. This game ha no end, the goal is to get the highest score.

This project was made for the Ubisoft contest 2023.

Technologies

Language
C#
Software & Ressources
Unity, Rider, Wwise
Unity Specifics
New Input System, Scriptable Object

My Role

In this project, I was responsible for overall management and assisted with various aspects. That being said, the tasks I carried out on my own included, among other things, a generic object management system incorporating factories and object pooling. I implemented the audio system and optimized it to avoid clusters of destruction or laser noise, for example. I also developed the mathematical algorithm for the procedural animation of turrets and target prediction for shooting.
